# VoyageAI Beta Definition

## Beta is complete when:

### 1. User can enter:
- [ ] Passport country
- [ ] Destination
- [ ] Dates (fixed or flexible)
- [ ] Budget

### 2. System returns:
- [ ] **GO / POSSIBLE / DIFFICULT** verdict
- [ ] Visa timeline + requirements
- [ ] True cost including visa & insurance
- [ ] Day-by-day itinerary

### 3. Experience guarantees:
- [ ] Shows visible progress within 3 seconds
- [ ] Never shows a blank screen
- [ ] Allows sharing a view-only link

---

## Out of Scope for Beta

- User accounts / authentication
- Booking integrations
- Real-time flight prices
- Collaboration / multi-user editing
- Activity images
- Drag-and-drop itinerary editing
- PDF export
- Mobile app

---

## Success Metrics for Beta

| Metric | Target |
|--------|--------|
| Time to first visible result | < 3 seconds |
| Time to complete result | < 45 seconds |
| Verdict clarity (user understands without explanation) | 90%+ |
| Cost accuracy (within 15% of actual) | 85%+ |
| Share rate (users who share their trip) | > 10% |

---

## Beta Launch Checklist

- [ ] Phase 1 complete (streaming, verdict, cost)
- [ ] Phase 2 complete (itinerary, action items)
- [ ] Phase 3 complete (share, analytics)
- [ ] Tested with 5 real trip scenarios
- [ ] No critical bugs in core flow
- [ ] Error states handled gracefully
