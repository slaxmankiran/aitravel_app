## Copy Guidelines

### Headlines
- Lead with certainty, not inspiration
- Use questions: "Can I go?" not "Dream destination"
- Numbers are trust signals: "87% certainty", "$2,430 true cost"

### CTAs
- Primary: "Check Trip Feasibility"
- Secondary: "View Full Plan"
- Share: "Share Trip"

### Error States
- Never blame user
- Always suggest next action
- Example: "Visa might take longer than your trip. Consider extending dates."
